Trang Chủ Lập Trình Java Spring Boot Phần Mềm

Tạo Ứng Dụng Web Với Spring Boot Sử Dụng Spring Tool

Spring Tool là bộ công cụ cho phép lập trình viên dễ dàng phát triển ứng dụng với Spring Framework. Spring Tool được hỗ trợ trên công cụ Eclipse, Visual Studio Code và Theia IDE. Trong bài viết này tôi sẽ hướng dẫn bạn từng bước để tạo một ứng dụng web sử dụng Spring Tool cho Eclipse.

Trước tiên bạn cần tải về bộ công cụ Spring Tool hoàn toàn miễn phí tại spring.io/tools

Tạo dự án Web với Spring Boot

Sau khi tải ứng dụng Spring Tool cho Eclipse và khởi động ứng dụng Spring Tool như hình bên dưới.

Để khởi tạo một ứng dụng mới, chọn menu File > New > Spring Starter Project

Từ cửa sổ New Spring Starter Project nhập vào thông tin dự án và chọn Next.

Từ cửa sổ New Spring Starter Project Dependencies chọn Thymeleaf, Spring Web và chọn Finish.

Sau khi hoàn thành các bước tạo ứng dụng mới như trên Spring Tool sẽ tạo một ứng dụng Spring Boot với cấu trúc như hình bên dưới.

Lập Trình Một Trang Web Đơn Giản với Spring MVC

Bước đầu tiên hãy tạo một package mới cho ứng dụng của bạn để chứa các class controller của Spring MVC.

Nhấp chuột phải vào package com.kynanglaptrinh.web và chọn New > Package

Từ cửa sổ New Java Package nhập vào tên package mới như hình bên dưới và chọn Finish.

Tiếp theo để tạo class controller nhấp chuột phải vào package com.kynanglaptrinh.web.controller và chọn New > Class

Từ cửa sổ New Java Class nhập vào tên class mới IndexController và chọn Finish

Viết code cho IndexController.java bạn vừa tạo như bên dưới. 

package com.example.web.controller;

import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;

@Controller
public class IndexController {
	
	@GetMapping("/")
	public String index(Model model) {
		model.addAttribute("message", "Chào mừng bạn đến với Kỹ Năng Lập Trình");
		return "index";
	}

}

Tiếp theo để tạo file HTML, nhấp phải vào thư mục templates và chọn New > File

Từ cửa sổ Create New File nhập vào tên file index.html và chọn Finish.

Viết code cho index.html bạn vừa tạp như sau. 

<!DOCTYPE HTML>
<html lang="en" xmlns:th="http://www.thymeleaf.org">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
    <title>Kỹ Năng Lập Trình</title>
</head>
<body>
<h1 th:text="${message}"></h1>
</body>
</html>

Biên dịch và khởi chạy ứng dụng Spring Boot trên Spring Tool

Để khởi chạy ứng dụng vừa tạo trên Spring Tool bạn mở file WebApplication.java chọn icon Run > Run As > Spring Boot App

Sau khi ứng dụng khởi chạy thành công bạn sẽ nhận được kết quả log của Spring Boot vào cửa sổ Console như sau.

Chú ý Spring Boot đã khởi chạy ứng dụng web ở cổng 8080. Để kiểm tra kết quả bạn khởi động trình duyệt và mở địa chỉ web http://localhost:8080/ và nhận được kết quả trang web như bên sau.
