Trang Chủ Lập Trình Java Spring Boot Phần Mềm

Hướng dẫn khởi tạo và đóng gói dự án Java bằng Maven trên Command Line

Để giúp đơn giản hóa quá trình khởi tạo một dự án Java, Maven cung cấp công cụ archetype. Một archetype có thể hiểu là một bản mẫu dự án Java được dựng sẵn để giúp lập trình viên tiết kiệm thời gian cấu hình một dự án mới. Trong bài viết này chúng ta sẽ tìm hiểu từng bước sử dụng Maven trên Command Line để khởi tạo một dự án Java đơn giản sau đó biên dịch và đóng gói dự án ra file .jar

Tạo dự án mới sử dụng mvn archetype:generate

Để khởi tạo dự án Java mới bạn cần cung cấp những thông tin sau

  • groupId
  • artifactId
  • archetypeArtifactId
  • interactiveMode
D:\>mvn archetype:generate -DgroupId=com.kynanglaptrinh -DartifactId=SimpleApp -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
