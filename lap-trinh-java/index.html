Maven là gì? Hướng dẫn cài đặt Maven trên Windows

Apache Maven là một công cụ cho phép lập trình viên chuẩn hóa việc quản lý mã nguồn của một dự án Java. Maven hỗ trợ lập trình viên từ các bước tạo lập dự án Java, quản lý thư viện được sử dụng trong dự án (hay còn gọi dependencies), quản lý phiên bản, đến việc biên dịch, kiểm thử (unit test) mã nguồn và đóng gói dự án.

Viết Chương Trình Java Hello World với Notepad và Command Line

Trong bài viết này tôi sẽ hướng dẫn bạn cách viết một chương trình Java đơn giản bằng cách sử dụng các công cụ có sẵn trên hệ điều hành Windows là Notepad và giao diện dòng lệnh Command Line.
